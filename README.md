# Harry Potter OWL Ontology

An OWL ontology modelling the Harry Potter universe, written in **Manchester Syntax** (`.omn`).

Covers core concepts such as characters, houses, spells, magical creatures, and their relationships — including properties like `isHeadOf`, house membership, and character roles.

## Files

| File | Description |
|------|-------------|
| `harry_potter_ontology.omn` | OWL ontology in Manchester Syntax |

## Usage

Open `harry_potter_ontology.omn` in [Protégé](https://protege.stanford.edu/) or any OWL-compatible tool.
